## Description

This workshop addresses a number of topics that play a role when natural language is interpreted and translated to Linked Data. In particular, it brings together researchers with a background in knowledge representation & automated reasoning and researchers working on NLP to discuss the following topics:

- **Attribution:**
  - How can we represent the source of information?
  - Should there be a fundamental difference between sources mentioned in text (quoted or cited authorities) and sources such as the author and publisher?
  - How do we distinguish information presented as a certain fact, a possibility, a prediction, etc.?

- **Domain modeling (for the financial domain):**
  - What do we need to model for end users of analyzed financial news?
  - How can we use domain knowledge to improve results from NLP?

The goal of the workshop is to establish a general model for representing the attribution of information (the source and certainty this source attributes to this information) and to outline the principles for creating a basic domain specific ontology that can be used in the NewsReader project.
