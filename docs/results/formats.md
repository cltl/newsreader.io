- [Grounded Annotation Format (GAF)](http://groundedannotationframework.org/): a framework to represent events in a formal context that can represent information from both textual and extra-textual sources

- [NLP Annotation Format (NAF)](https://newsreader.vossen.info/newsreader/naf/): The NAF format is linguistic annotation format designed for complex NLP pipelines. NAF combines strengths of the Linguistic Annotation Framework (LAF) and the NLP Interchange Format.

- [Simple Event Model (SEM)](https://newsreader.vossen.info/newsreader/sem/): SEM is an ontology to describe events in terms of “**who** did **what** with **what** to **whom**,**where**, and **when**?”
